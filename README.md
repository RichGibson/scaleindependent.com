# Scale Independent

Personal site of Rich Gibson. Maps, gigapans, code, culture, and the occasional poem.

Live at [scaleindependent.com](https://scaleindependent.com).

## Stack

- [Hugo](https://gohugo.io/) static site generator
- [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme
- Deployed on Cloudflare Pages

## Local development

```
hugo server
```

Then open http://localhost:1313.

## Content

- `content/posts/` — main blog posts (maps, gigapans, culture, personal essays)
- `content/til/` — today I learned (code notes by topic)
- `content/photo_projects/` — photo essays
- `content/family/` — family writing
